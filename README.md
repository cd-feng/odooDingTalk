# Odoo16系统集成钉钉

## 介绍
本项目是一套基于`Odoo`平台的开源的一个集成阿里钉钉应用，仓库中所有模块均为开源模块


## 安装
在安装模块前，务必先安装额外的三方库`（requirements.txt）`文件中，安装方式示例：

 `pip3 install -r $PATH/requirements.txt`  注意$PATH为你的实际路径!
